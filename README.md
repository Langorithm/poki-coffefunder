# poki-coffefunder

Solo Poki game project built with Godot 4.x.

## Structure

```
src/              Godot 4.x project
📐 design/        Game design + architecture notes (Obsidian)
💼 business/      Planning, milestones, monetization (Obsidian)
📓 devlog/        Development journal (Obsidian)
```

## Dev Setup

1. Open `src/` in Godot 4.x
2. Open this folder as an Obsidian vault
